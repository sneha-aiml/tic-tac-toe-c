# Tic-Tac-Toe Game in C

A console-based two-player Tic-Tac-Toe game built in C.
Submitted as part of the Programming for Problem Solving 
course at JBIET (2025-26).

## Features
- Two-player mode (X and O)
- Move validation — no overwriting occupied cells
- Win detection across all 8 combinations
- Draw condition detection
- Clean modular code structure

## How to Run
gcc tictactoe.c -o tictactoe
./tictactoe

## Tech Used
- Language: C
- Compiler: GCC
- Editor: VS Code
